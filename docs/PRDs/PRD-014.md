# One recommendation before we close Phase 2

Once PRD-014 passes, I'd like us to create a **v1.0.0-beta Git tag** and treat it as a permanent milestone. From that point onward:

- **CSE** (the framework) should evolve cautiously with backward compatibility in mind.
- **CPE** (the Claire Performance Engine) can evolve rapidly through RFDs and experimentation.
- **CSM** (the Claire Speech Model) can iterate independently as new training techniques emerge.

That separation lets you innovate aggressively where it matters—speech intelligence—without destabilizing the framework that developers depend on. I think that's the strongest long-term foundation for the project we've been designing together.
