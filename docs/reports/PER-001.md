# 6. Known Limitations

The following implementations are intentionally minimal:

### Meaning Pass

Current implementation performs deterministic semantic placeholder generation.

Future semantic reasoning will replace the placeholder without changing the public interface.

---

### Intent Pass

Current implementation performs deterministic punctuation-based intent classification.

Future intent reasoning will extend the implementation without changing the interface.

---

### Performance Planning Pass

Current implementation uses deterministic rule-based planning.

Future planning models may replace the implementation while preserving compatibility.
